# Instacart Market Basket Analysis

This project contains an exploratory data analysis (EDA) notebook for the Instacart dataset. The goal is to understand the structure of the data, explore key tables, and clean missing values before further analysis.

## Project Overview

The notebook analyzes several Instacart-related datasets to learn about:
- customer orders
- products
- aisles
- departments
- order-product relationships

The analysis focuses on inspecting the data, identifying missing values, and understanding how different tables connect.

## What is included

- A Jupyter notebook named EDA.ipynb
- Data exploration using pandas
- Summary inspection with .info() and .head()
- Missing value checks and basic cleaning steps

## Main steps in the notebook

1. Import required libraries
2. Load the Instacart datasets
3. Explore the structure of each table
4. Identify missing values
5. Clean and understand missing data in the products, orders, and order_products tables

## Dataset focus

The notebook works with these datasets:
- orders
- products
- departments
- aisles
- order_products

## Notes

This project is primarily an EDA exercise and is intended to build a foundation for future analysis, such as customer purchasing patterns or product recommendations.
